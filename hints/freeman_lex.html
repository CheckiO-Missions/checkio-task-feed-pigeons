The food may not be enough to feed fed + new, but there two case possible here. The food remains may be bigger than presented pigeons and enough to feed some part of new pigeons for the first time, so the total number of fed pigeons will increase and the general number of pigeons you fed this minute (and the result of function) will be equal food.

The second possible variant is when food remains is smaller then fed - not even enough to feed already presented pigeons, but the whole number of fed pigeons are already fed at least once, so the result of function in this case is fed.

In short, you are seeking the maximum between two values - pigeons you may feed this minute(food remains) and which are already fed. 

result = max(food, fed)
So, how my final solution should look like?

You just need to wrap your code into function expression and return result expression.

Here is the whole solution:

def checkio(food: int) -> int:

    fed = 0
    new = 1
    while food >= fed + new:
        fed += new
        food -= fed
        new += 1
    
    return max(food, fed)

In this solution the recursion is used. If not enough food, you return values in the same cases as in solution from hints. If there is enough food, you call function again with updated values. 

def checkio(food: int, all_pig: int = 1, fed: int = 0) -> int:

    if food <= fed:
        return fed
    if food <= all_pig:
        return food
        
    return checkio(food - all_pig, 2 * all_pig - fed + 1, all_pig)

Sequence of the pigeons presented for the every minute is 1, 3, 6, 10, 15 ... Sequence of the fed pigeons for the every minute is 1, 4, 10, 20, 35 ... It's the a(i) = i*(i+1)*(i+2)/6 sequence. Put a(i) = food and get the cubic equation: i*(i+1)*(i+2) - 6*n = 0 which is possible to solve with Cardano's method. 

from math import sqrt


def checkio(food: int) -> int:

    alpha = (3*food - sqrt(9*(food**2)-1/27)) ** (1/3)
    beta = (3*food + sqrt(9*(food**2)-1/27)) ** (1/3)
    minute = (alpha + beta - 1) // 1 + 1
    
    fed = (minute-1)*minute/2
    last_feed = (minute**2 - 1)*minute / 6
        
    return max(food - last_feed, fed)
